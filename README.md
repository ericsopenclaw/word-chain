# Word Chain - 文字接龍遊戲

一個好玩的中文文字接龍遊戲，支援單人練習和雙人對戰模式。

## 🎮 功能特色

- **單人模式**：練習模式，無時間限制
- **雙人對戰**：兩位玩家輪流接龍
- **計時模式**：可設定時間限制挑戰
- **詞庫驗證**：檢查輸入是否為有效詞彙
- **歷史記錄**：顯示所有接龍記錄
- **計分系統**：統計雙方得分

## 📝 遊戲規則

1. 第一位玩家輸入一個詞
2. 下一位玩家必須輸入一個以上一個詞最後一個字開頭的新詞
3. 不能重複使用已經用過的詞
4. 如果找不到合適的詞，可以選擇「放棄」
5. 最後無法接龍的玩家輸掉遊戲
